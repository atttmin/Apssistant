APssistant Instructions

Main features

  • Custom shortcut for HUD color picker (CS5 Only)
  • Custom shortcut for foreground color picker (Any version)
  • Color picker shortcut sticky
  • Color picker shortcut map Alt
  • Change Step Forward / Backward shortcut
  • Autosave/alarm
  • Double-clik to quick start Photoshop (CS5 Only)
  • Lock input method to English
  • Shortcut tray notificaion
  • Important Notice

Instructions

  • # HUD color picker[ ^ ]

    This feature only works for Photoshop CS5, you need to enable OpenGL support in Photoshop.
    How to enable OpenGL:
    Edit > Preferences > Preformance... > Enable OpenGL Drawing
    This is not fully supported by some video cards (The checkbox is grey), you can try to modify your registry or update your video cards driver. (or buy a new one.)

  • Foreground color picker[ ^ ]

    Photoshop CS5 or any further version you can go here to set your shortcut:
    keyboard shortcut and menu > shortcut for (Tools)> foreground color picker
    Photoshop CS4 or any previous version (at least 6) please choose your Photoshop version. Then you can opening or save color picker with it.

    Notice:
    This shortcut also allows to implement confirm function, which is "one key operation".
    If you want to use foreground color picker shortcut in Photoshop CS4 or any previous version (at least 6), you need to meet the following conditions:

    1. Non-full-screen mode (Foreground color require clickable)
    2. Useing the brush tool or Pencil tool
    3. Useing system default theme (not classics or any custom)

    If you cannot use this feature in non-system default theme, please go to PsCS5++\Imagesearch and replace with your reference image.

  • Color picker "Sticky"[ ^ ]

    HUD color pikcer checkd "Sticky" will don't need hold space for changing hue.
    foreground color picker checkd "Sticky" will enter on key up (looks like Alt Eyedropper Tool )

  • Color picker shortcut Alt mapping[ ^ ]

    This function will take color picker and Alt Eyedropper Tool combined, a total of three modes:

    1. Disable Alt mapping
      Disable this function
    2. Shortcut map Alt (Compat Mode)
      Use the color picker trigger of CS4 or any previous version.
    3. # Shortcut map Alt (CS5 Mode)
      Only display with Photoshop CS5 or any further version, you should enter the color picker shortcut built-in .CS5 or any further version.
  • Change Step Forward / Backward shortcut[ ^ ]

    This function is convenient for users who frequent Reinstallation and modify shortcut, the 4 default shortcut keys will cycle replacement as follows:
    Step Backward
    from Alt+Ctrl+z to Ctrl+z
    proof colors
    from Ctrl+y to Alt+Ctrl+z
    Step Forward
    from Shift+Ctrl+z to Ctrl+y
    undo/redo
    from Ctrl+z to Shift+Ctrl+z

  • Autosave[ ^ ]

    You can choose auto save mode or reminder mode.

  • # Double click tray to Launch Photoshop[ ^ ]

    Only works for CS5 (also works for x64), still need further development for pervious versions.

  • Lock input method to English[ ^ ]

    This feature is for non-English users to avoid shortcuts confliction.

  • Disable tray notificaion[ ^ ]

    This program display tray notification of some feature you can disable them here.
